# bill2myprint-ops
Ansible script to deploy bill2myprint (OFRF) service.

Commands
---------

Run on VM test:
```
./ofrfsible
```
or
```
./ofrfsible --vm --test
```

Run on VM prod:
```
./ofrfsible --prod
```
or
```
./ofrfsible --vm --prod
```

Run on kubernetes TKGI dev:
```
./ofrfsible --k8s --dev
```
